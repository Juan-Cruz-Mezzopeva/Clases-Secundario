1. La Estructura Invisible Obligatoria

Toda página web real necesita iniciar con estas cuatro cajas fundamentales que le avisan al navegador cómo interpretar el documento.

Etiqueta y Código HTML ¿Para qué sirve? ¿Cómo se ve en el navegador?
<!DOCTYPE html>
<html lang="es">		 Avisa al navegador que esta es una página web moderna (HTML5) y configura el idioma oficial en español para buscadores. Invisible. Opera de forma interna.
<head> ... </head> Guarda configuraciones técnicas ocultas. Adentro va la etiqueta <title>, que define el nombre de la pestaña superior. Cambia el título de la pestaña de arriba de la pantalla.
<body> ... </body> Es el cuerpo principal. Absolutamente todo lo que el usuario ve (imágenes, textos, botones) debe ir aquí metido. Es todo el lienzo o espacio visible de la pantalla actual.

2. Estructuración de Textos y Títulos Semánticos

El orden y tamaño de los textos le otorgan jerarquía a la información para que sea fácil de leer.

Código HTML ¿Para qué sirve? Previsualización en Pantalla
<h1>Título Principal</h1> Encabezado de máxima importancia. Solo debe haber uno solo por cada página web.

Título Principal
<h2>Subtítulo</h2> Se usa para dividir la página en grandes bloques, apartados o temas lógicos independientes.

Subtítulo de Sección
<p>Este es un párrafo...</p> Crea bloques de texto fluido. Agrega automáticamente un espacio en blanco debajo para separarse del siguiente párrafo.

Este es un párrafo de texto común donde se escribe la biografía o descripciones del sitio.

3. Listas y Agrupación de Ítems

Ideales para agrupar canciones favoritas, integrantes de bandas, redes sociales o menús de navegación.

Código HTML ¿Para qué sirve? Previsualización en Pantalla 
<ul>
  <li>Playstation</li>
  <li>Xbox</li>
</ul>
 Unordered List (Lista Desordenada): Agrupa elementos utilizando viñetas de puntos, sin un orden numérico específico.
  • Playstation
  • Xbox
 
<ol>
  <li>Primer Paso</li>
  <li>Segundo Paso</li>
</ol>
 Ordered List (Lista Ordenada): Añade números secuenciales automáticos (1, 2, 3...) antes de cada ítem en pantalla.
  1. Primer Paso
  2. Segundo Paso

4. Atributos Esenciales: Modificadores de Enlaces e Imágenes

Los atributos se inyectan dentro de la etiqueta inicial para modificar su comportamiento o enlazar archivos externos.

Etiqueta con Atributos ¿Qué hace cada atributo? Resultado Interactivo 
<a href="https://google.com" 
   target="_blank">
   Ir a Google
</a>
 href: Indica la dirección URL exacta de destino.
target="_blank": Obliga al navegador a abrir el enlace en una pestaña nueva.
Ir a Google 
<img src="https://picsum.photos/60" 
     alt="Imagen" 
     class="foto-perfil">
 src: Ruta o enlace directo a la imagen en internet.
alt: Texto alternativo por si la imagen se rompe.
class: Apodo único para poder darle estilos estéticos con CSS.

5. Formularios de Entrada (Interactividad de Usuario)

Las etiquetas que recolectan información para iniciar sesión, enviar datos o escribir en un chat.

Estructura de Código ¿Para qué sirve? Elemento Real Funcional 
<form>
  <label>Usuario:</label>
  <input type="text" placeholder="Ej: Juan">
  <button type="button">Entrar</button>
</form>
 <form>: Caja contenedora del formulario.
<input>: Campo libre de escritura.
placeholder: Texto guía gris de fondo.

6. Estructuración en Tablas de Datos

Sirve para organizar matrices rígidas de datos organizadas estrictamente por filas y columnas.

Código de la Tabla Desglose de Etiquetas Renderización Visual 
<table>
  <tr>
    <th>Día</th> <th>Materia</th>
  </tr>
  <tr>
    <td>Lun</td> <td>Web 101</td>
  </tr>
</table>
 <table>: Contenedor general.
<tr>: Fila horizontal (Row).
<th>: Celda de encabezado (Bold).
<td>: Celda de dato común (Data).
Día Materia
Lun Web 101
